## 経緯と目的
- V最協決定戦のカスタムを見ていて、同時接続数がチャンネル登録者数やVtuber界隈の知名度と純粋な比例関係ではなさそうだった
- 急に同時接続数を取得してみたくなったので本番前日にコード書き始めた
- 本来は出場者全員の同時接続数を取得したかった
- しかし準備不足のため、今回は比類なき証明のメンバーのみの同時接続数を取得することにした
- 今回は[今後](#future)のためのジャブ的な感じにならばええなと思っている
